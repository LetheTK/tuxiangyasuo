# 图片压缩网站

## 项目说明
这是一个简单的在线图片压缩工具网站。用户可以上传图片,系统会自动压缩图片大小,同时尽量保持图片质量,然后提供下载链接。

## 特性
- 支持拖拽上传
- 支持多种图片格式（JPG、PNG、WEBP）
- 智能压缩算法
- 实时预览
- 文件大小限制（最大10MB）
- 深色模式支持
- 响应式设计
- 错误处理和用户反馈

## 使用说明
1. 点击上传区域或将图片拖拽到上传区域
2. 等待图片处理完成
3. 预览压缩效果
4. 点击"下载压缩后的图片"按钮保存

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API

## 开发说明
1. 克隆项目
2. 打开 index.html 即可运行
3. 修改 css/style.css 自定义样式
4. 修改 js/compressor.js 自定义压缩参数
